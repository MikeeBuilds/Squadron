# Squadron Skill

You have access to the `squadron` CLI tool to communicate with the human team.

## When to use
- After completing a coding task.
- When you need to report a bug or blocker.
- To move a Jira ticket status to "Done".

## Commands

### Reporting Progress
`squadron report --msg "I fixed the login bug." --ticket "PROJ-101"`

### General Chat
`squadron report --msg "I am starting the backtest analysis now."`
